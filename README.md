## 🎯 Current Specialization & Focus

I am currently hyper-focused on writing **bulletproof frontend foundations** by mastering:

- **Semantic HTML & Accessibility (A11Y + WCAG):** Going beyond basic tags to implement machine-readable structures (like ISO 8601 time markers) and ensuring inclusive experiences for screen readers.
- **BEM Methodology:** Writing predictable, modular, and self-contained CSS architectures that eradicate style regression.
- **Maintainability & Clean Code:** Engineering centralized **"Design Token Vaults"** (`:root`) to eliminate magic numbers and mapping out styles using clean, highly-documented code sections.
- **Fluid Responsive Design:** Moving away from rigid breakpoints and mastering fluid typography/layouts using native CSS functions like `clamp()`.
- **Atomic Commits:** I treat git history like a well-written book. I separate structural blueprints (HTML) from visual systems (CSS) to keep repositories clean and rollback-safe.
